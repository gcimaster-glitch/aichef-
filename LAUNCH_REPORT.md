## 🎯 プロジェクト概要

### **AIChef とは？**

AIChefは、**忙しい家族のための献立作成＆買い物リスト自動生成アプリ**です。

**主な機能**:
1. **AI献立自動生成**: 家族構成、予算、アレルギーを考慮した献立を自動作成
2. **買い物リスト**: 献立から必要な食材を自動集計
3. **レシピデータベース**: 295件の高品質レシピ（主菜・副菜・汁物）
4. **栄養バランス**: 主菜のタンパク源、副菜、汁物を自動組み合わせ
5. **アレルギー対応**: 特定食材を自動除外
6. **予算管理**: 1人あたりの予算を設定可能

## 🎓 学びのポイント（振り返り）

### **技術的な学び**

1. **Cloudflare Workers環境の制約**
   - Node.jsモジュールが使えない
   - Web Crypto APIをフォールバックとして実装
   - bcryptjsが動作しない場合の対処法

2. **外部キー制約の重要性**
   - `member_sessions` → `members`の参照関係
   - データ整合性の保証
   - エラーメッセージからの原因特定

3. **JWTトークン設計**
   - アクセストークン（短期）+ リフレッシュトークン（長期）
   - payload設計（household_id + member_id）
   - 署名検証の重要性

4. **Git/GitHubの活用**
   - コミットメッセージの重要性
   - バージョン管理のメリット
   - クラウドバックアップ

---

1. **段階的な開発**
   - Phase A+B+C → Phase 1 → ローンチ
   - 各フェーズで明確な成果
